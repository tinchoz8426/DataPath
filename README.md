# 📊 DataPath · Roadmap interactivo de Ingeniería de Datos

**DataPath** es una guía visual e interactiva para convertirte en Ingeniero de Datos desde cero. Está estructurada en **15 fases** que cubren desde los fundamentos de Linux y Python hasta arquitecturas modernas como Data Lakehouse, pasando por SQL, Spark, Airflow, Docker y Cloud.

Este proyecto no es solo un listado de temas, sino una **herramienta de seguimiento activo** que te permite marcar tu progreso, guardar tus dudas y mantener la motivación con un sistema de logros y un temporizador Pomodoro integrado.

*Nota: La estructura del contenido, la documentación y el texto de este roadmap han sido generados y optimizados con la asistencia de Inteligencia Artificial (IA), garantizando un enfoque pedagógico, moderno y detallado.*

## ✨ Características principales

- **📈 Progreso persistente** – Marca cada ítem como completado. Todo se guarda automáticamente en el `localStorage` de tu navegador.
- **❓ Sistema de dudas** – Haz clic en el icono `❓` para marcar cualquier concepto que no te quede claro. Los ítems con dudas se resaltan en rojo y puedes filtrarlos para repasarlos.
- **🏆 Logros y gamificación** – Desbloquea insignias al alcanzar hitos (primer ítem, 50% de progreso, completar todas las fases, etc.). Los logros se celebran con confeti.
- **⏱️ Pomodoro integrado** – Un temporizador flotante con ciclos de enfoque/descanso que cuenta tus pomodoros completados y los guarda día a día.
- **🔖 Modo lectura** – Oculta los checkboxes y las distracciones para revisar el contenido sin interrupciones.
- **📤 Compartir progreso** – Genera un mensaje con tu porcentaje completado y compártelo en Twitter o LinkedIn (con copia al portapapeles).
- **📱 Diseño responsivo** – Adaptado para escritorio, tablet y móvil, con navegación lateral (escritorio) y horizontal (móvil).
- **⚡ Sin dependencias externas pesadas** – Todo el código corre directamente en el navegador (HTML, CSS y JavaScript vanilla), solo usa CDNs para fuentes e iconos.

## 🗺️ Estructura del Roadmap

| Fase | Tema |
|------|------|
| `0` | Preparación del entorno (VS Code, Git, Python, Docker, etc.) |
| `1` | Linux para datos (comandos básicos, scripting, permisos) |
| `2` | Git (control de versiones y colaboración) |
| `3` | Python (fundamentos, colecciones, APIs, logging) |
| `4` | SQL (consultas avanzadas, CTE, ventanas, optimización) |
| `5` | Modelado de datos (normalización, estrella, copo de nieve, SCD) |
| `6` | Pandas (limpieza, transformación y análisis) |
| `7` | ETL y ELT (pipelines, calidad, idempotencia) |
| `8` | Docker (imágenes, contenedores, compose) |
| `9` | Airflow (DAGs, operadores, orquestación) |
| `10` | Spark (procesamiento distribuido, DataFrames, optimización) |
| `11` | Cloud (AWS: S3, EC2, RDS, Glue, Athena, EMR) |
| `12` | dbt (transformaciones, tests, documentación) |
| `13` | Kafka (streaming, productores, consumidores) |
| `14` | Data Lakehouse (Medallion, Delta Lake / Iceberg) |

Cada fase incluye su objetivo, los módulos con sus ítems y un proyecto práctico asociado.

## 🛠️ Tecnologías utilizadas

- **HTML5** – Estructura semántica.
- **CSS3** – Estilos personalizados con temática oscura/neón, efectos de vidrio y transiciones suaves.
- **JavaScript (ES6+)** – Lógica de interacción, persistencia, gamificación y temporizador.
- **localStorage** – Guardado automático del progreso, logros y dudas.
- **Google Fonts** – Tipografías *Orbitron* y *Rajdhani*.
- **Font Awesome** – Iconografía.
- **Canvas Confetti** – Animaciones de celebración al desbloquear logros.

## 📂 Instalación y uso local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/DataPath.git
   ```
2. Abre el archivo `index.html` en tu navegador favorito.

¡Empieza a marcar tu progreso! Todos los datos se guardarán automáticamente en tu navegador sin necesidad de servidores, bases de datos ni instalaciones complejas.
