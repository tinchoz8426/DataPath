# 📊 DataPath · Roadmap interactivo de Ingeniería de Datos

**DataPath** es una guía visual e interactiva para convertirte en Ingeniero de Datos desde cero. Está estructurada en **fases** que cubren desde los fundamentos de Linux y Python hasta arquitecturas modernas como Data Lakehouse, pasando por SQL, Spark, Airflow, Docker y Cloud.

Este proyecto no es solo un listado de temas, sino una **herramienta de seguimiento activo** que te permite marcar tu progreso, guardar tus dudas, gestionar tus tareas con un tablero Kanban, completar proyectos prácticos por cada módulo y mantener la motivación con un sistema de logros y un temporizador Pomodoro integrado.

*Nota: La estructura del contenido, la documentación y el texto de este roadmap han sido generados y optimizados con la asistencia de Inteligencia Artificial (IA), garantizando un enfoque pedagógico, moderno y detallado.*

## ✨ Características principales

### Roadmap interactivo
- **📈 Progreso persistente** – Marca cada ítem como completado. Todo se guarda automáticamente en el `localStorage` de tu navegador.
- **❓ Sistema de dudas** – Haz clic en el icono `❓` para marcar cualquier concepto que no te quede claro. Los ítems con dudas se resaltan en rojo y puedes filtrarlos para repasarlos.
- **🏆 Logros y gamificación** – Desbloquea insignias al alcanzar hitos (primer ítem, 50% de progreso, completar todas las fases, etc.). Los logros se celebran con confeti.
- **⏱️ Pomodoro integrado** – Un temporizador flotante con ciclos de enfoque/descanso que cuenta tus pomodoros completados y los guarda día a día.
- **🔖 Modo lectura** – Oculta los checkboxes y las distracciones para revisar el contenido sin interrupciones.
- **📤 Compartir progreso** – Genera un mensaje con tu porcentaje completado y compártelo en Twitter o LinkedIn (con copia al portapapeles).

### Tablero Kanban
- **🗂️ 5 columnas de flujo de trabajo** – Pendiente → Investigando → Aprendido → Documentado → Publicado. Organiza tus tareas de estudio según su estado real.
- **🖱️ Drag & Drop** – Arrastra y suelta tarjetas entre columnas para cambiar su estado de forma visual e intuitiva. Las columnas se resaltan al arrastrar sobre ellas.
- **➕ Crear, editar y eliminar tareas** – Cada tarea tiene título, descripción opcional y estado. Modales dedicados para cada acción con confirmación de eliminación.
- **🧹 Limpiar tablero** – Botón para eliminar todas las tareas de golpe, con modal de confirmación para evitar accidentes.
- **🔢 Contadores por columna** – Cada columna muestra en tiempo real cuántas tareas contiene.
- **⌨️ Atajos de teclado** – `Escape` para cerrar cualquier modal, `Ctrl+Alt+N` para crear una tarea nueva rápidamente.
- **🔔 Notificaciones toast** – Feedback visual no intrusivo al crear, mover, editar o eliminar tareas.
- **💾 Persistencia en localStorage** – Todas las tareas se guardan automáticamente; se restauran al recargar la página.

### Proyectos prácticos
- **🛠️ Proyectos por cada módulo** – Cada una de las 17 fases del roadmap tiene proyectos sugeridos para aplicar lo aprendido en la práctica. No son obligatorios: sentite libre de adaptarlos o crear los tuyos.
- **📝 Detalle por proyecto** – Cada proyecto incluye su nombre, una descripción con los pasos sugeridos, las tecnologías involucradas (tags visuales) y un enlace de referencia cuando aplica.
- **✅ Marcar proyectos como completados** – Checkbox por cada proyecto con progreso global independiente del roadmap principal.
- **❓ Dudas por proyecto** – Podés marcar dudas a nivel de proyecto individual y filtrarlos para repasar.
- **🏆 Logros propios** – Sistema de logros desbloqueable exclusivo para proyectos (primer proyecto completado, 25%, 50%, 75%, 100%, etc.) con celebración de confeti.
- **📊 Barra de progreso independiente** – Seguimiento visual del porcentaje de proyectos completados, separado del roadmap.
- **🔀 Colapsar/expandir módulos** – Botón para colapsar o expandir todos los módulos de proyectos a la vez.
- **📤 Compartir progreso de proyectos** – Genera un mensaje con tu porcentaje de proyectos completados para compartir en redes sociales.
- **⏱️ Pomodoro integrado** – El mismo temporizador flotante disponible mientras trabajás en tus proyectos.

### Compartidas
- **📱 Diseño responsivo** – Adaptado para escritorio, tablet y móvil. El roadmap usa navegación lateral (escritorio) y horizontal (móvil). El Kanban adapta sus columnas al tamaño de pantalla. Los proyectos también se adaptan completamente.
- **⚡ Sin dependencias externas pesadas** – Todo el código corre directamente en el navegador (HTML, CSS y JavaScript vanilla), solo usa CDNs para fuentes e iconos.

## 🗺️ Estructura del Roadmap

| Fase | Tema |

## 📂 Instalación y uso local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tinchoz8426/DataPath.git
   ```
2. Abre el archivo `index.html` en tu navegador favorito para usar el **Roadmap**.
3. Abre `kanban.html` para acceder al **Tablero Kanban**.
4. Abre `proyectos.html` para ver y marcar los **Proyectos Prácticos** de cada módulo.

¡Empieza a marcar tu progreso y gestionar tus tareas! Todos los datos se guardarán automáticamente en tu navegador sin necesidad de servidores, bases de datos ni instalaciones complejas.
